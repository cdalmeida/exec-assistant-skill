# Calibration Log

_Rolling 14-day window of MIT completion. Used to recalibrate the user's daily target._

Default target: **3–5 MITs/day**. Adjusted only after a 14-day window shows a sustained pattern, and only with the user's explicit confirmation.

## How to read this

- Observation period: first 14 days after setup. No adjustments proposed.
- After day 14: check the rolling average. If ≤2 → propose lowering target or diagnosing blockers. If ≥5 clean → offer to raise target. Otherwise keep.
- Re-check every 14 days.
